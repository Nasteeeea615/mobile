# Mock режим для тестирования без бекенда

## Как использовать

Mock режим позволяет полностью протестировать приложение без запуска бекенда. Все данные хранятся в памяти приложения.

### Включение/выключение Mock режима

Откройте файл `mobile/src/services/mockApi.ts` и измените константу:

```typescript
// Включить mock режим (работа без бекенда)
export const MOCK_MODE = true;

// Выключить mock режим (работа с реальным бекендом)
export const MOCK_MODE = false;
```

## Тестовые данные для входа

### SMS код
При входе используйте любой номер телефона и код: **1234**

### Сценарии тестирования

#### 1. Новый пользователь (Клиент)
1. Введите любой номер телефона (например: +79991234567)
2. Введите код: 1234
3. Заполните форму регистрации
4. Вы попадете на главный экран создания заказа

#### 2. Новый пользователь (Исполнитель)
1. Введите любой номер телефона
2. Введите код: 1234
3. На экране регистрации нажмите "Хочу стать исполнителем"
4. Заполните форму регистрации исполнителя
5. Вы попадете на экран доступных заказов

#### 3. Существующий пользователь
После первой регистрации, если вы введете тот же номер телефона и код 1234, вы автоматически войдете в систему.

## Что работает в Mock режиме

✅ Авторизация (вход по SMS)
✅ Регистрация клиента
✅ Регистрация исполнителя
✅ Создание заказов
✅ Просмотр списка заказов
✅ Принятие заказов (для исполнителя)
✅ Завершение заказов
✅ Оплата заказов
✅ Создание тикетов поддержки
✅ Просмотр истории обращений
✅ Редактирование профиля

## Особенности Mock режима

- Все данные хранятся в памяти и сбрасываются при перезапуске приложения
- Задержка ответов ~800ms для имитации реальных сетевых запросов
- В консоли отображаются все mock запросы с префиксом `[MOCK API]`
- SMS код всегда: **1234**

## Переход на реальный бекенд

Когда бекенд будет готов:

1. Откройте `mobile/src/services/mockApi.ts`
2. Измените `export const MOCK_MODE = false;`
3. Убедитесь что в `mobile/src/services/api.ts` правильно указан `API_URL`
4. Перезапустите приложение

## Отладка

Все mock запросы логируются в консоль:
```
[MOCK API] POST /auth/send-sms { phoneNumber: '+79991234567' }
[MOCK] SMS код: 1234
[MOCK API] POST /auth/verify-sms { phoneNumber: '+79991234567', code: '1234' }
```
